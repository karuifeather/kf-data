# Finance Tracker

## Project Overview

Finance Tracker is a personal finance management application developed to help users gain control over their spending habits, set budget goals, and monitor progress toward financial health.

## Project Architecture

The application is structured as a three-tier architecture, with a React frontend for a seamless user experience, a Node.js server for handling requests and business logic, and MongoDB as the database to store user data securely.

## Design Philosophy

The app was designed with user-centric principles, focusing on simplicity and ease of use. A minimalistic interface helps users focus on key features without distraction, while responsive design ensures usability across devices.

## Technologies Used

- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Problems Faced

One significant challenge was managing data synchronization between the frontend and backend in real time, especially for updating user expenses. The solution was to implement a web socket connection to push data updates as they occur.

## Results

Since its release, Finance Tracker has received positive feedback from users for its intuitiveness and reliability. It successfully tracks expenses and helps users stay on budget, fulfilling its intended purpose.
